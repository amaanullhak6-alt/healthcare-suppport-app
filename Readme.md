Healthcare Support Portal

Project Overview

Healthcare Support Portal is a simple web application designed to help NGOs and healthcare volunteers collect patient information and provide quick assistance.

The application allows patients to submit their details and symptoms through an easy-to-use form. It also includes a basic AI-inspired automation feature that generates a symptom summary and assigns a priority level based on the symptoms entered.

---

Features

Patient Support Form

Patients can submit:

- Full Name
- Age
- Phone Number
- Symptoms / Health Concerns

AI-Powered Symptom Analysis

The application automatically:

- Generates a patient symptom summary
- Assigns a Priority Level (Low, Medium, High)
- Suggests an appropriate action

Automation Logic

Low Priority

General symptoms or non-urgent concerns.

Suggested Action:
Monitor symptoms and stay hydrated.

Medium Priority

Symptoms such as:

- Fever
- Headache
- Body Pain

Suggested Action:
Contact a healthcare volunteer for guidance.

High Priority

Symptoms such as:

- Chest Pain
- Breathing Problems
- Shortness of Breath

Suggested Action:
Seek immediate medical attention.

AI / Automation Feature

The application includes a simple symptom analysis system.

Based on keywords entered by the patient, the system:

1. Generates a symptom summary.
2. Determines a priority level.
3. Recommends the next action.

This demonstrates how basic AI-assisted triage can support NGOs and healthcare volunteers in managing patient requests more efficiently.
